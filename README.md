# Neon Tetris

一個可直接在瀏覽器開啟的俄羅斯方塊小遊戲。

## 我幫你定的版本
- 主題：霓虹街機風
- 平台：純前端單頁網頁
- 核心玩法：經典俄羅斯方塊
- 節奏：每消除 10 行升 1 級，速度逐步加快
- UI：主遊戲區、下一塊預覽、分數 / 等級 / 行數 / 最佳分數、操作說明

## 操作
- ← / →：左右移動
- ↑：旋轉
- ↓：快速下落
- Space：瞬間落下
- P：暫停 / 繼續

## 開始方式
直接開啟 `index.html` 即可遊玩。

## 檔案結構
- `index.html`：畫面結構
- `styles.css`：視覺樣式
- `script.js`：遊戲邏輯
